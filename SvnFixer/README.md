# SvnFixer# SvnFixer

SvnFixer 是一个用于修复和优化 SVN 仓库的 .NET 10 工具。小小工具，不足挂齿，方便自己，方便他人

## 功能特性

- 自动检测并修复常见的 SVN 仓库问题（目前仅支持清除工作队列，有需要再补充）
- 详细的日志输出，便于问题追踪
- AOT编译
